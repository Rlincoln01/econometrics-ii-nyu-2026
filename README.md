### Notes on materials & sharing

This repository is **public**. To avoid unintentionally redistributing copyrighted material (e.g., textbooks, publisher PDFs, or instructor-owned slides/handouts), the repo tracks **structure + documentation + syllabi**, while other PDFs are kept **untracked** via `.gitignore`.

If you are enrolled in the course and need access to restricted materials, please use the official NYU course channels or email the instructors/TA.
